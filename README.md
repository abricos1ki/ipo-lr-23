# Лабораторная работа №16. Создание и базовая настройка приложений Django

**Вариант: 22**
**Тема магазина: интернет-магазин товаров для йоги**

## Суть приложения

Учебный проект на Django, демонстрирующий базовую настройку проекта
и приложения для дальнейшей разработки интернет-магазина товаров для йоги
(коврики, аксессуары, одежда и товары для дома для практики йоги).

На данном этапе реализована только структура проекта и три простые
страницы, отдающие текстовый ответ (`text/plain`), без моделей и базы
данных товаров — это будет добавлено в следующих лабораторных работах.

## Настройки проекта (settings.py)

- **База данных:** SQLite (`django.db.backends.sqlite3`) — используется по умолчанию, файл `db.sqlite3` создаётся автоматически после миграций.
- **Часовой пояс:** `Europe/Minsk`
- **Язык интерфейса:** `ru` (русский)
- **Приложение `shop`** подключено в `INSTALLED_APPS`.

## Установка и запуск

1. Клонировать репозиторий:
   ```bash
   git clone <ссылка на репозиторий>
   cd yoga_shop_lab16
   ```

2. Создать и активировать виртуальное окружение:

   **Windows:**
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```

   **Linux / macOS:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Применить миграции:
   ```bash
   python manage.py migrate
   ```

5. Запустить сервер разработки:
   ```bash
   python manage.py runserver
   ```

6. Открыть в браузере: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
